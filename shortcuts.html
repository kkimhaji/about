IntelliJ 단축키

단축키 내용
Alt + Enter 코드에서 오류가 발견된 경우 문제에 대한 수정 제안 목록 표시
F2 F2를 사용하여 다음 오류, 경고, 제안으로 점프
⌘1 또는 Alt+1 도구 창
Esc 에디터에 포커스 위치
⌘E 또는 Ctrl+E 최근 실행 파일 확인
⌘B 또는 Ctrl+B 코드 안에서 다른 코드로 이동
Alt+F7 어떤 항목이 사용된 모든 곳을 표시
Ctrl Ctrl 최근 실행 목록 구성 표시, 모든 항목 검색 및 실행 가능
⌥↑ / ⌥↓
Ctrl + W / Shift + Ctrl + W 		커서 근처의 코드 선택 영역을 확대, 축소 가능
⌘/ 또는 Ctrl + / 해당 줄 주석 처리
⇧⌘⏎ 또는 Shift+Ctrl+Enter 현재 구문 완성
⌥⌘L 또는 Ctrl+Alt+L 파일 서식 지정
⌃T 또는 Shift+Ctrl+Alt+T 사용 가능한 리팩토링 옵션이 표시
⇧⌘A 또는 Shift+Ctrl+A Intellij IDEA에서 모든 액션을 검색 가능.
Shift Shift
  • ⌘E / Ctrl+E 대신 사용 가능.
  • 모든 항목을 검색할 수 있는 검색 상자가 열림.
